picoRTOS scheduler API

PICORTOS_STACK_COUNT(x)

Computes stack count from array

Parameters

xa picoRTOS_stack_t array

PICORTOS_DELAY_SEC(x)

Converts seconds to picoRTOS_tick_t

Parameters

xa value in seconds

PICORTOS_DELAY_MSEC(x)

Converts milliseconds to picoRTOS_tick_t

Parameters

xa value in milliseconds

PICORTOS_DELAY_USEC(x)

Converts microseconds in picoRTOS_tick_t

Parameters

xa value in microseconds

picoRTOS_invalidate_dcache(addr, n)

If DEVICE_HAS_SOFTWARE_CACHE_COHERENCY, invalidates cache at addr up to addr+n, otherwise, do nothing (hardware cache snooping)

Parameters

addrThe memory base address to invalidate
nThe size (in bytes) to invalidate

picoRTOS_flush_dcache(addr, n)

If DEVICE_HAS_SOFTWARE_CACHE_COHERENCY, flushes cache at addr up to addr+n, otherwise, do nothing (hardware cache snooping)

Parameters

addrThe memory base address to flush to
nThe size (in bytes) to flush
